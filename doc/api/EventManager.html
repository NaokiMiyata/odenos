EvenManager

EventManager is a class of implementing pub-sub communication between RemoteObject instances
Followings are brief description of EventManager's REST API

REST APIs

  • GET <base_uri>/property
  • PUT <base_uri>/property
  • GET <base_uri>/settings/event_subscriptions
  • PUT <base_uri>/settings/event_subscriptions/<subscriber_id>
  • GET <base_uri>/settings/event_subscriptions/<subscriber_id>

GET <base_uri>/property

get Object property.

[Request]:
  • Body : none
[Response]:
  • Status Code : 200
  • Body : ObjectProperty

PUT <base_uri>/property

update Object property.

[Request]:
  • Body : ObjectProperty
[Response]:
  • Status Code : 200
  • Body : ObjectProperty

GET <base_uri>/settings/event_subscriptions

get Event Subscription list.
<subscriber_id> is EventSubscription.subscriber_id

[Request]:
  • Body : none
[Response]:
  • Status Code : 200
  • Body : dict{subscriber_id, EventSubscription}

PUT <base_uri>/settings/event_subscriptions/<subscriber_id>

Event Subscription settings make a new registration.

[Request]:
  • Body : EventSubscription
[Response]:
  • Status Code : 200
  • Body : EventSubscription

GET <base_uri>/settings/event_subscriptions/<subscriber_id>

get Event Subscription. is EventSubscription.subscriber_id

[Request]:
  • Body : none
[Response]:
  • Status Code : 200
  • Body : EventSubscription
